read

() Array

Defined in server/api/users.js:12

Gets all users. Has both a local (api.users.get) and a remote (api/users/read.json) version.

Returns:

Array: JSON object containing the user records with user id, name, email, company, country, created, modified, and _id.

Example:

GET http://mobilent.com/api/users/read.json
